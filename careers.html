Software Developer (Server Team)

The role of the Server Tools Developer includes the following tasks.

  • - Develop software products used to maintain millions of SIM/USIM cards deployed on the network.
  • - Deploy and maintain current deployed servers.
  • - Maintain current code base including hundreds of thousands of lines of code.

The Software Tools Developer reports to the Software Products Development Team Leader.
